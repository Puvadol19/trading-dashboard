# Trading Dashboard — Chrome Side Panel Extension

## วิธีติดตั้ง

1. เปิด Chrome แล้วไปที่ `chrome://extensions/`
2. เปิด **Developer mode** (toggle มุมบนขวา)
3. กด **"Load unpacked"**
4. เลือกโฟลเดอร์ `chrome-extension` ทั้งโฟลเดอร์นี้
5. Extension จะปรากฏในแถบ toolbar

## วิธีใช้งาน

1. กดไอคอน Trading Dashboard ใน Chrome toolbar
2. Side Panel จะเปิดทางด้านขวาของหน้าจอ
3. Dashboard จะโหลด `https://trading-dashboard-dpzh.vercel.app/` อัตโนมัติ
4. ใช้ปุ่ม **Reload** เพื่อรีเฟรช
5. ใช้ปุ่ม **Open Tab** เพื่อเปิดในแท็บใหม่

## หมายเหตุ

- Chrome Side Panel ต้องการ Chrome version 114+
- ปรับความกว้าง panel ได้โดยลากขอบซ้ายของ panel
